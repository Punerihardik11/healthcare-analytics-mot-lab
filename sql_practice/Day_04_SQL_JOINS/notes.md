# Day 04 - SQL JOINS

## Topics Practiced
- INNER JOIN
- LEFT JOIN
- ON clause
- Matching keys
- Relational databases

## Key Learnings
- JOIN combines related tables
- INNER JOIN keeps matching rows only
- LEFT JOIN keeps all rows from left table
- IDs are commonly used as join keys
- Row counts should be validated after joins

## Realizations
- Real healthcare systems store data in separate tables
- JOINs are critical for healthcare analytics
- Most business dashboards depend on table relationships
