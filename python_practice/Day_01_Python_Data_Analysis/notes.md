# Day 01 - Python Data Analysis Basics

## Topics Practiced
- groupby()
- mean()
- sum()
- size()
- agg()
- rename()
- isnull()

## Key Learnings
- groupby() works like SQL GROUP BY
- mean() calculates averages
- sum() calculates totals
- size() counts rows
- agg() performs multiple aggregations
- rename() renames columns
- isnull().sum() checks missing values

## SQL vs Pandas Mapping
- GROUP BY → groupby()
- AVG() → mean()
- SUM() → sum()
- COUNT(*) → size()
